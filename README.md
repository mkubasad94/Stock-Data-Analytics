# Stock-Data-Analytics

## Source details:
- Source: CSV files
- Frequency: Daily
- Volume: 20 years historical data
- Update: Daily append

### Dataset has below columns:
| Column Name        | Data Type | Description            |
| ------------------ | --------- | ---------------------- |
| Date               | Date      | Trading date           |
| Symbol             | String    | Stock symbol           |
| Series             | String    | Security series        |
| Prev Close         | Decimal   | Previous closing price |
| Open               | Decimal   | Opening price          |
| High               | Decimal   | Highest price          |
| Low                | Decimal   | Lowest price           |
| Last               | Decimal   | Last traded price      |
| Close              | Decimal   | Closing price          |
| VWAP               | Decimal   | Volume weighted price  |
| Volume             | Long      | Traded quantity        |
| Turnover           | Decimal   | Total traded value     |
| Trades             | Integer   | Number of trades       |
| Deliverable Volume | Long      | Delivered quantity     |
| %Deliverble        | Decimal   | Delivery percentage    |
